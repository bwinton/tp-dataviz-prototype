Get the source CSV files by downloading:
https://testpilot.mozillalabs.com/testcases/beta/interface_small.tar.gz

Edit csv-to-json.py to choose the number of users you want to get and add any filters you want (e.g. drop scroll events)

Run python csv-to-json.py

Output is beta_ui-data.json, which is input to the dataviz prototype.
